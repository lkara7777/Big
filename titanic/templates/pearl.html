Calculations - Pearl curve


The Pearl or Logistic Curve states that:

L: maximum value of our quantity, for this example, the maximum orders of the product .
t: time, for our case it is per month
a,b: values that we must calculate so as to achieve the best possible calculations in order to minimize the Mean Absolute Percentage Errors.(MAPE)
MAPE:we try to minimize the absolute value of the predicted values for y minus the real values, that means that we predict values close to the real.
In order to conclude to : y=75800/(1+ 404.4*exp(-0.306*t)), we need to find the best values for L, a, b.




The main logic is to guess a value for the L near to the biggest real value, for example L=16. After, you calculate a,b and with these values you predict the values of y according to the formula. Also, you fill the column F and you find the average absolute error.
You do the same process for many values of L, greater and smaller , and you keep the values of L,a,b in case of the smallest MAPE. After that, you have ready your equation of curve.
